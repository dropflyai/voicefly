Test Scenarios

Manual Testing Steps

  1. Click "Reset Welcome Flag" above
  2. Make sure you have 0 phone employees (delete via Supabase or employees page)
  3. Navigate to /dashboard
  4. Wait 3 seconds - Maya should auto-open
  5. Close Maya and reload page - should NOT auto-open again
  6. Click "Set to 25 Hours Ago" above
  7. Reload /dashboard - should auto-open again

Browser Console Commands

// Check current state localStorage.getItem('maya_welcome_seen') // Reset (force auto-open on next visit) localStorage.removeItem('maya_welcome_seen') // Set to now (prevent auto-open) localStorage.setItem('maya_welcome_seen', new Date().toISOString()) // Set to 25 hours ago (allow auto-open) const old = new Date(Date.now() - 25 * 60 * 60 * 1000) localStorage.setItem('maya_welcome_seen', old.toISOString())

Expected Behavior

  • New User (0 employees, no flag): Auto-opens after 3 seconds on /dashboard
  • Returning User (<24h): Does NOT auto-open
  • Old Returning (>24h, still 0 employees): Auto-opens again
  • Has Employees: Never auto-opens (not on step 1)
  • Other Pages: Never auto-opens (only /dashboard)
